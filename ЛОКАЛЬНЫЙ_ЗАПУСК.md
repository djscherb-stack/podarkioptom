# Запуск на компьютере (локально)

Чтобы проверить изменения, не выгружая в интернет.

---

## Быстрый запуск «всё в одном»

В терминале выполните:

```bash
cd /Users/dmitriy/ProAnalitik
./run-local.sh
```

Скрипт сам:
- освободит порты 8000 и 5173;
- создаст venv (если нет) и установит зависимости backend;
- установит зависимости frontend и **соберёт** его (актуальный интерфейс);
- запустит сервер на http://localhost:8000;
- откроет браузер с автовходом (без формы логина).

**Не закрывайте окно терминала** — сервер работает, пока скрипт не завершён. Остановить: **Ctrl+C**.

## Если не запускается

### Нет venv
```bash
cd /Users/dmitriy/ProAnalitik
python3 -m venv venv
source venv/bin/activate
pip install -r backend/requirements.txt
```

### Нет frontend/dist
```bash
cd /Users/dmitriy/ProAnalitik/frontend
npm install
npm run build
```

### Порт 8000 занят
```bash
lsof -ti:8000 | xargs kill -9
```

### Ручной запуск
```bash
cd /Users/dmitriy/ProAnalitik
source venv/bin/activate
cd backend
uvicorn app:app --host 0.0.0.0 --port 8000
```
