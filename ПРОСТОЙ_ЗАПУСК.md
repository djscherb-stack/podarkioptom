# Самый простой способ выложить сайт в интернет (бесплатно)

**Перед началом:** проверь, что Excel-файлы лежат в папке `data/`.

## Шаг 1. Зарегистрируйся на GitHub

1. Зайди на https://github.com
2. Нажми **Sign up**, создай аккаунт (бесплатно)

---

## Шаг 2. Залей проект на GitHub

1. Открой терминал на компьютере (где лежит ProAnalitik)
2. Выполни по порядку:

```bash
cd /Users/dmitriy/ProAnalitik

# Создать репозиторий
git init
git add .
git commit -m "Аналитика"
```

3. На GitHub нажми **New repository**, назови `proanalitik`, не ставь галочки, нажми **Create**
4. Скопируй команды, которые GitHub покажет (типа "push an existing repository")
5. Вставь их в терминал и нажми Enter:

```bash
git remote add origin https://github.com/ТВОЙ_ЛОГИН/proanalitik.git
git branch -M main
git push -u origin main
```

(Подставь свой логин вместо ТВОЙ_ЛОГИН)

---

## Шаг 3. Разверни на Render

1. Зайди на https://render.com
2. Нажми **Get Started for Free**, войди через GitHub
3. Нажми **New** → **Web Service**
4. Выбери репозиторий **proanalitik**
5. Оставь всё как есть (Render сам увидит Docker)
6. Внизу нажми **Create Web Service**
7. Подожди 3–5 минут, пока собирается
8. Готово! Сайт будет по адресу типа: `https://proanalitik-xxx.onrender.com`

---

## Как добавлять новые Excel-файлы

1. Положи новый файл в папку `data/` (рядом с backend и frontend)
2. В терминале:
```bash
cd /Users/dmitriy/ProAnalitik
git add .
git commit -m "Новые данные"
git push
```
3. Render сам обновит сайт за 2–3 минуты
4. Открой сайт и нажми ⟳ (обновить)

---

**Важно:** бесплатный тариф на Render «засыпает» после 15 минут без заходов. Первый заход может открываться 30–60 секунд.
